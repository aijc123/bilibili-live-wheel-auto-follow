Bilibili Live userscript

B站独轮车 + 自动跟车

给 B 站/哔哩哔哩直播间用的民间弹幕助手。它可以循环发送提前写好的弹幕，自动跟上正在刷屏的句子，也能把右侧评论区换成更清爽的 Chatterbox Chat。

Greasy Fork 安装 GitHub 源码

支持 Tampermonkey、Violentmonkey 等用户脚本管理器。项目基于 LAPLACE Chatterbox fork 修改。

主要功能

独轮车循环发送

多套模板、每行一句、按间隔发送，支持随机颜色、随机间隔、随机字符和长弹幕自动拆分。

自动跟车

观察短时间内重复出现的弹幕，达到阈值后自动跟一句。内置稳一点、正常、热闹三档预设。

Chatterbox Chat

接管右侧评论区，显示 iMessage 风格气泡，支持礼物、醒目留言、舰队等事件和自定义 CSS。

粉丝牌禁言巡检

读取粉丝牌关联房间并分类展示限制、无法确认、主播注销和正常结果，可复制报告或同步摘要。

替换与 AI 规避

支持云端和本地替换规则，发送失败时可检测敏感词并尝试替换后重试。

同传和烂梗库

接入 Soniox 语音识别，可自动发送识别或翻译结果，也能搜索、复制和发送常用梗。

安装使用

  1. 先安装 Tampermonkey 或 Violentmonkey。
  2. 打开 Greasy Fork 脚本页，点击安装。
  3. 进入 B 站直播间，点击右下角的「弹幕助手」。
  4. 需要接管评论区时，在「设置」里开启 Chatterbox Chat。
